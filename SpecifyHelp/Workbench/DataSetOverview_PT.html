Visão geral do conjunto de dados

Dados são inseridos, editados, organizados e guardados em um Conjunto de Dados.

O tipo de dado que será inserido irá determinar o tipo de item de ação a ser utilizado para a criação do Conjunto de Dados.

Para inserir dados de notas de campo ou etiquetas selecione a ação Conjunto de dados novo para criar um conjunto de dados vazio. Os nomes das colunas do Conjunto de dados novo serão selecionados dos campos dentro do Specify 6, o que garante que qualquer dado inserido no WorkBench será importado de forma correta para o Specify 6. O título da coluna poderá ser renomeado, mas ainda manterá um link para o campo orginal do Specify. Estes links são chamados 'mapas'.

Use a ação Importar dados para importar dados de um MS Excel ou arquivo CSV. O WorkBench não apenas importa os dados, mas também ajuda a mapear os cabeçalhos das colunas existentes para os campos do Especify 6 com um recurso de 'mapeamento automático' que lê os cabeçalhos das colunas de importação e mapeia automaticamente para quaisquer campos correspondentes. Qualquer cabeçalho de coluna restante não mapeado, pode ser mapeado manualmente. Os títulos das colunas podem ser mapeados para diferentes campos Especify posteriormente, usando o Editar mapeamento do conjunto de dados.

Os conjuntos de dados exibem os títulos das colunas importadas em vez dos nomes dos campos, pois são mais familiares.

A ação Importar imagens vincula imagens a registros individuais em um novo conjunto de dados. Este item de ação é especialmente útil ao inserir dados digitalizados, como imagens ou notas de campo.

O WorkBench também pode vincular imagens individualmente a cada linha dentro de um conjunto de dados existente.

Os conjuntos de dados são editados como um Grid ou um Formulário. Cada visualização oferece funcionalidade exclusiva:

Edição de Grid

Painel de pesquisa

Georreferenciamento com BioGiomancer

Mapeamento com o Google Earth

Ferramenta de conversão de georreferência

Exportar registros selecionados

Editar formulário

Renomear colunas

Reorganizar formato do formulário na área de trabalho.

 

Um conjunto de dados inteiro ou linhas selecionadas podem ser exportados para uma planilha do MS Excel usando o Exportar conjunto de dados.

Um conjunto de dados pode ser descrito, bem como nomeado usando a caixa de diálogo de propriedades do conjunto de dados.
